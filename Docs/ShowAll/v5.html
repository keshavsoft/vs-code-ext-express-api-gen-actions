What Happens When You Click get?

The get command generates the ShowAll endpoint architecture automatically. The extension creates backend layers for reading all records from the journals table.

1

Generate ShowAll Folder

The extension automatically creates the ShowAll folder inside the journals module.

2

Create controller.js

The controller handles requests and sends JSON responses back to the browser.

3

Create service.js

The service layer contains business logic and reads JSON table data.

4

Register Endpoint

The extension automatically creates the GET endpoint inside end-points.js.

ShowAll Endpoint Execution Flow

The request travels through multiple Express.js layers before returning JSON data.

Browser

User calls ShowAll endpoint.

routes.js

Main Express router receives the request.

end-points.js

Generated ShowAll endpoint gets executed.

controller.js

Processes request and response handling.

service.js

Reads journals JSON records from storage.

JSON Response

Records returned back to frontend application.
